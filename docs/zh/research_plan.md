# 研究计划

## 核心问题

与无情绪表达的 Avatar 和真实教师基线相比，由情绪模型驱动的 3D 教学 Avatar 是否能够提升学习者的参与感、自然感、社会临场感和整体学习体验？

## 拟定研究重点

本项目不只是构建一个通用的 LLM Avatar 演示，而是聚焦于具备情绪感知能力的教学 Avatar。系统使用情绪模型，将教学上下文和学生状态转换为面部表情、视线、点头、手势以及语音相关行为。

## 候选情绪模型

- PAD/VAD 模型：适合连续的表情和动作控制。
- OCC 模型：适合教学语境中的认知评价，例如学生犯错后的鼓励，或取得进展后的表扬。
- LLM 辅助估计：适合从对话上下文中抽取结构化的情绪状态。

第一版实现大概率会以 PAD/VAD 作为核心表示，并使用 LLM 输出作为状态估计方法。

## 基线条件

- 纯文本 LLM 教师。
- 无情绪表达的 3D Avatar 教师。
- 基于规则的情绪化 Avatar 教师。
- 情绪模型驱动的 Avatar 教师。
- 真实教师教学作为参考条件。

## 评估指标

- 通过前测和后测衡量学习增益。
- 自我报告的理解程度。
- 参与度和动机。
- 社会临场感。
- 信任感和友好感知。
- 面部表情和动作的自然度。
- 端到端延迟和模块级延迟。

## 作品集目标

该实现也应该作为 AI Agent 方向的作品集项目。系统应展示工具调用、记忆、工作流编排、追踪，以及与实时 3D 客户端的集成能力。
